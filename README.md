## 📦 Installation

1. Download the theme
2. Go to WordPress Dashboard > Appearance > Themes
3. Click "Add New" then "Upload Theme"
4. Choose the downloaded zip file and click "Install Now"
5. Activate the theme

## 🛠️ Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
